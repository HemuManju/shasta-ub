============
Installation
============


Stable release
--------------

To install shasta, run this command in your terminal:

.. code-block:: console

    $ pip install shasta

This is the preferred method to install shasta, as it will always install the most recent stable release.

If you don't have `pip`_ installed, this `Python installation guide`_ can guide
you through the process.

.. _pip: https://pip.pypa.io
.. _Python installation guide: http://docs.python-guide.org/en/latest/starting/installation/


From sources
------------

The sources for shasta can be downloaded from the `Github repo`_.

You can either clone the public repository:

.. code-block:: console

    $ git clone https://github.com/ub-nsf-hsi/shasta-ub.git


Install the requirements:

.. code-block:: bash

   pip install -r requirements.txt

Install using python:

.. code-block:: console

    $ python setup.py install


.. _Github repo: https://github.com/ub-nsf-hsi/shasta-ub

Configure your SHaSTA installation and simulator settings as needed.
